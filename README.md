# Dino_Swing

### Description:
Ucode project: develop minigame like Google T-REX game. 

### Usage:
```
1. git clone https://github.com/trohalska/Dino_Swing Dino
2. cd Dino
3. ./gradlew build
4. ./gradlew run
```
